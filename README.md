
# 개발배경

1. 본 앱은 PHP, MySQL, Bootstrap, jQuery를 이용하여 제작한 웹 앱입니다.
2. Cordova로 패키징하고 gitgub에서 형상관리를 하였습니다.
3. 도서관리 시스템으로 회원의 계정과 책을 관리할수 있는 웹 앱입니다.
